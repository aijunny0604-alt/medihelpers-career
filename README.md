# 메디헬퍼스 의료 커리어 플랫폼

## Purpose

의료인에게는 비공개 커리어 상담과 검증된 채용정보를, 병원에는 채용공고 광고와 전담 헤드헌팅을 제공하는 웹 플랫폼입니다.

## Current State

- React + Vite 기반 반응형 웹사이트
- 홈, 채용정보, 인재정보, 헤드헌팅, 광고센터, 서비스 소개 페이지 제공
- 채용 필터, 관심공고 저장, 상담 신청, 광고 상품 선택 및 결제 요청 흐름 구현
- 실제 회원 인증, 서버 저장, PG 결제는 다음 단계에서 연결

## Current Rules

- 의료인은 무료로 이용하고 개인정보는 공개 전에 반드시 별도 동의를 받습니다.
- 병원 광고 상품과 헤드헌팅 서비스에서 매출을 만듭니다.
- 표시되는 의료인 프로필은 익명 예시이며 실제 개인정보를 노출하지 않습니다.
- 결제 완료로 오인할 수 있는 표현은 PG 연동 전까지 사용하지 않습니다.

## Run

```bash
npm install
npm run dev
npm run build
```
