# 메디헬퍼스 의료 커리어 플랫폼

의사는 검증된 초빙정보와 비공개 이직 상담을 받고, 병원은 의사 초빙광고와 동의 기반 의사 소개를 이용하는 의사 전문 헤드헌팅 플랫폼입니다.

## 공개 사이트

- [메디헬퍼스 Sites 공개본](https://medihelpers-career.junnyai.chatgpt.site)
- [GitHub 저장소](https://github.com/aijunny0604-alt/medihelpers-career)
- [GitHub Pages 미러](https://aijunny0604-alt.github.io/medihelpers-career/)
- 최신 Sites 공개 버전: v141

## 현재 구현

- 홈, 채용정보, 헤드헌팅, 매칭리포트, 광고센터, 멤버십, 서비스 소개 페이지
- 초빙유형·진료과·지역·근무조건 중심의 의사 전용 초빙정보 검색
- 의사 비공개 이직 상담과 병원 의사 채용 의뢰를 전담 헤드헌터로 연결
- 채용 검색·필터, 관심 공고, 상세 공고 모달
- 의사·병원용 건별 열람권과 월 멤버십 비교
- 프리미엄 공고와 익명 인재 상세정보의 유료 권한 안내
- 상담, 광고, 멤버십 결제 안내 요청 접수
- 의사·병원별 3단계 간편상담과 상담번호 발급
- 의료인·병원 프론트엔드 전용 가입 신청 초안 폼(이름·휴대폰·이메일·비밀번호, 병원은 기관명·담당 역할 추가): 개인정보를 저장하지 않고(완료·초기화 시 즉시 삭제) `가입 양식 확인 완료`까지만 처리하며 실제 계정을 만들지 않음 — 실제 인증·본인확인·서버 계정 생성은 법무 확정 후 연결
- 예전 `/professions`(의사 이직상담)는 헤드헌팅(`/headhunting`)으로 통합되어 리다이렉트
- 데스크톱·태블릿·모바일 반응형 화면
- 찜한 병원·인재를 조건별로 비교하고 헤드헌터 상담으로 전달하는 매칭 리포트
- 병원 브랜드와 포지션 정보를 분리한 프리미엄 광고 쇼케이스
- 모달 대신 독립 페이지로 운영되는 병원 초빙공고 등록 흐름과 브랜드 이미지 1장·병원 갤러리 분리 업로드
- 병원·의사 로그인 상태에 따라 공고/이력서, 문의·상담, 결제·사용이력, 알림과 회원정보를 다르게 보여주는 역할별 마이페이지
- 모든 회원에게 공통으로 제공되는 아이디·로그인 도움, 회원정보 수정, 알림 수신 설정과 계정 보안 안내

## 핵심 정책

- 기본 검색과 공개 요약은 무료로 제공합니다.
- 상세 급여·비공개 포지션·검증 인재 상세정보와 소개 요청은 유료 권한으로 제공합니다.
- 병원 결제는 의사의 원본 개인정보 구매가 아닙니다. 후보 의사의 명시적 동의 후 필요한 정보만 전달합니다.
- 현재 결제 화면은 요청 접수 단계이며 실제 카드 과금은 PG와 서버 검증 연결 후 활성화합니다.

## 실행

```bash
npm install
npm run dev
npm run build
```

## 배포와 동기화

- 검증이 끝난 변경은 문서와 함께 GitHub `main` 브랜치에 푸시합니다.
- GitHub Pages 공개본은 `main` 푸시 후 GitHub Actions로 자동 배포됩니다.
- OpenAI Sites 공개본은 같은 커밋으로 별도 저장·배포해 GitHub와 버전을 맞춥니다.
- 로컬 파일 변경만으로 GitHub에 실시간 자동 푸시되지는 않습니다. 작업 완료와 검증 후 의도적으로 커밋·푸시합니다.
