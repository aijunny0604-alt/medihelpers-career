# 참고자료

## Purpose

시장 조사와 외부 참고자료를 기록하되 제품 문구와 디자인을 복제하지 않기 위한 공간입니다.

## Current State

- 메디게이트: 진료과·지역·채용유형 중심 탐색, 초빙정보, 커리어 상담, 광고 상품 구조 참고
- 메디헬퍼스 차별점: 전담 헤드헌터의 인간적인 상담과 검증된 소규모 큐레이션

## Current Rules

- 참고 URL, 확인일, 배운 점만 기록합니다.
- 타사의 상표, 카피, 이미지, 화면 배치를 복사하지 않습니다.
