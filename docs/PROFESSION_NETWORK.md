# 보건의료 직군 통합 네트워크

## 원칙

하나의 플랫폼에 모든 직군을 모으되 검색조건과 커뮤니티를 섞지 않습니다. 회원 인증, 결제, 상담, 병원 계정은 공통으로 사용하고 직군별 채용정보와 콘텐츠는 독립적으로 운영합니다.

## 직군 허브

- 의사·치과의사·한의사
- 간호사·간호조무사·조산사
- 약사·한약사
- 방사선사
- 임상병리사
- 물리치료사·작업치료사·언어재활사
- 치과위생사·치과기공사
- 응급구조사·보건의료정보관리사·영양사·안경사·요양보호사 등

## 공통 기능

- 면허·자격과 소속기관 인증
- 익명 프로필과 공개범위 관리
- 채용검색, 관심공고, 간편상담
- 건별 열람권과 직군별 멤버십
- 병원 광고, 후보 소개, 면접·채용 상태관리

## 직군별 데이터 예시

- 의사: 진료과, 당직, 진료범위, 보수와 인센티브
- 간호: 병동·부서, 교대형태, 임상경력, 전문간호 분야
- 방사선: CT·MRI·혈관조영·치료 등 모달리티와 장비경험
- 임상병리: 진단검사·병리·채혈·생리기능·수혈 분야
- 재활: 치료분야, 환자군, 도수·신경계 경험, 인센티브

## 커뮤니티 출시 순서

1. 직군 인증과 오픈 알림
2. 급여·근무표·면접정보 등 구조화된 익명 데이터
3. 검증회원 질문과 실무 경험 게시판
4. 교육·보수교육·자격 콘텐츠
5. 게시글에서 상담과 채용으로 연결

초기에는 의사 채용을 운영하고 다른 직군은 파트너 모집 상태로 표시합니다. 실제 공급 없이 예시 공고를 대량 게시하지 않습니다.
