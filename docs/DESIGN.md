# DESIGN

## Purpose

메디헬퍼스만의 브랜드와 반복 UI 기준을 정의합니다.

## Current State

### Concept — Human-led Medical Career Concierge

대형 포털처럼 정보량만 강조하지 않고, 실제 헤드헌터가 끝까지 동행한다는 신뢰를 전면에 둡니다.

### Visual System

- Navy `#10213F`: 전문성·신뢰
- Blue `#1263E8`: 연결·주요 행동
- Mint `#20B894`: 케어·검증·성공
- Surface `#F5F8FC`: 복잡한 채용 정보를 편하게 읽는 배경
- 둥근 카드, 얇은 경계선, 넉넉한 여백을 사용합니다.

### Navigation

- 채용정보: 의료인용 공고 탐색
- 인재정보: 병원용 익명 인재풀 소개
- 헤드헌팅: 의료인/병원 상담
- 광고센터: 병원용 광고 상품·주문
- 서비스 소개: 신뢰·절차·연락처

### Interaction

- 모든 주요 버튼은 실제 페이지, 전화, 이메일 또는 신청 흐름으로 연결합니다.
- 채용공고 카드는 빈 공간을 포함한 카드 전체를 상세보기 클릭 영역으로 사용하고, 관심공고 버튼만 독립 동작으로 분리합니다.
- 카드 전체 클릭은 키보드 초점과 Enter·Space 입력을 지원하며 명확한 초점 테두리를 표시합니다.
- 모바일에서는 메뉴를 접고 하단에 상담 전환을 유지합니다.
- 성공 메시지는 사용자가 무엇을 제출했는지 구체적으로 알려줍니다.
- 스크롤 진입 요소는 짧은 흐림과 아래 방향 이동에서 감속하며 정착하는 관성형 모션을 사용합니다.
- 첫 화면에만 약한 시차 효과를 적용하고, 스크롤 진행 상태는 상단의 얇은 선으로 표시합니다.
- PC의 마우스 휠·트랙패드 스크롤에는 짧은 관성과 감속을 적용해 페이지 자체가 부드럽게 따라오게 합니다.
- 모바일 터치 스크롤과 모달 내부 스크롤은 운영체제의 기본 동작을 유지합니다.
- `prefers-reduced-motion` 환경에서는 스크롤 등장·시차·진행 애니메이션을 비활성화합니다.

### Simplicity Rules

- 첫 화면의 핵심 행동은 `채용 찾기 / 비공개 상담 / 공고 등록` 세 가지로 제한합니다.
- 장식용 숫자와 사실 근거가 약한 통계 띠는 사용하지 않습니다.
- 카드에 표시하는 조건은 진료과, 지역, 근무일, 급여 순으로 고정합니다.
- 광고센터는 상단과 모바일 하단에 항상 노출하되 일반 채용공고와 혼동되지 않게 표시합니다.
- 광고 공고는 별도 광고 등급으로 관리하며, 로고가 등록된 경우 로고를 카드의 중심 브랜드 요소로 크게 표시합니다.
- 로고가 없는 광고 공고는 병원명을 워드마크처럼 강조하고, 추천·집중채용 등급에 따라 배경과 테두리의 강도를 구분합니다.
- 광고 카드의 빛 효과는 스크롤 진입 시 한 번만 실행하고 반복 재생하지 않습니다.
- 집중채용은 목록 최상단과 가장 강한 1회성 포커스 모션을, 추천 광고는 다음 순위와 중간 강도 모션을 사용합니다.

## Current Rules

- 의료인 사진을 장식 목적으로 남용하지 않습니다.
- 익명 인재카드는 식별 가능한 개인정보를 포함하지 않습니다.
- 애니메이션은 정보의 등장 순서를 돕는 범위에서만 사용하고 반복·회전·큰 시차 효과는 사용하지 않습니다.
