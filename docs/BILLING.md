# BILLING

## Purpose

병원 광고 상품과 안전한 결제 흐름을 정의합니다.

## Current State

화면에는 상품 선택과 결제 요청 단계가 구현되어 있습니다. 실제 카드 청구는 PG 가맹점 승인 후 연결합니다.

## Products

- 베이직 공고: 30일, 검색 노출 — 99,000원
- 추천 공고: 30일, 상단 추천·강조 — 249,000원
- 집중 채용: 45일, 최상단·배너·전담 컨설턴트 — 499,000원
- 전담 헤드헌팅: 별도 계약, 성공보수형

가격은 시장 검증용 초안이며 부가세 표시와 계약 조건을 출시 전 확정합니다.

## Payment Flow

1. 병원이 상품과 공고 정보를 입력
2. 서버가 주문번호와 최종 금액 생성
3. 토스페이먼츠 결제창 호출
4. 서버가 승인 API로 금액·주문 검증
5. 웹훅으로 결제 상태 재확인
6. 운영자 검수 후 공고 게시
7. 취소·환불 시 게시 상태와 결제 상태 동기화

## Current Rules

- 브라우저가 보낸 금액을 신뢰하지 않습니다.
- 결제 완료와 공고 게시 완료를 분리합니다.
- 의료광고·채용표현 검수 절차를 둡니다.
- 환불 기준, 게시기간, 부가세, 세금계산서 정책을 결제 전에 고지합니다.
