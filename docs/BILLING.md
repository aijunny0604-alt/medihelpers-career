# BILLING

## 수익 모델

메디헬퍼스는 광고, 정보 열람권, 채용 멤버십, 헤드헌팅 수수료를 분리해 판매합니다. 무료 이용자는 시장을 탐색할 수 있고 실제 의사결정에 필요한 검증 정보와 연결 권한은 유료로 제공합니다.

## 의료인용 상품

- 공고 1건 열람권: 3,900원 / 건
- Doctor Pass: 19,900원 / 월
- 유료 권한: 상세 급여, 정확한 근무시간, 비공개 병원·포지션, 커리어 상담

## 병원용 상품

- 인재 소개 요청권: 49,000원 / 건
- Hospital Recruit Pass: 149,000원 / 월
- 유료 권한: 검증된 익명 인재 상세정보, 소개 요청, 신규 인재 알림, 채용 컨설턴트 지원
- 의료인의 연락처나 원본 개인정보 자체를 판매하지 않습니다. 후보자의 명시적 동의가 확인된 소개 절차에서 필요한 정보만 제공합니다.

## 광고 상품

- 베이직 광고: 99,000원
- 추천 광고: 249,000원
- 집중 채용: 499,000원
- 광고 구매와 인재 개인정보 열람 권한은 별도입니다.

## 결제 흐름

1. 회원 유형과 자격을 확인합니다.
2. 서버가 상품, 금액, 주문번호를 확정합니다.
3. PG 결제창에서 결제를 요청합니다.
4. 서버 승인 API와 웹훅으로 결제 상태를 검증합니다.
5. 결제 완료 후 만료일과 사용 횟수가 있는 권한을 발급합니다.
6. 환불·취소 시 미사용 권한을 회수하고 감사 기록을 남깁니다.

## 전환 전략

- 먼저 진료과·지역·근무형태·익명 기본조건을 충분히 무료로 보여줍니다.
- 사용자가 특정 공고나 후보를 선택한 순간에만 잠긴 정보와 건별 가격을 제시합니다.
- 의료인은 공고 6건 이상, 병원은 소개 4건 이상이면 월 패스가 더 저렴하다는 사실을 계산기로 투명하게 안내합니다.
- 허위 마감, 가짜 이용자 수, 숨겨진 비용 같은 다크패턴은 사용하지 않습니다.
- `detail_open`, `unlock_cta`, `plan_select`, `checkout_request` 단계를 측정해 실제 전환 데이터를 기준으로 문구와 상품을 개선합니다.

## 현재 상태

웹 화면에는 상품 비교와 결제 안내 요청이 구현되어 있습니다. 실제 카드 과금과 자동 권한 발급은 PG 가맹점 계약, 서버 인증, 웹훅 검증을 연결한 뒤 활성화합니다.
