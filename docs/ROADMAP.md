# ROADMAP

## Purpose

시장 검증부터 실제 결제 플랫폼까지의 출시 순서를 정의합니다.

## Current State

### Phase 1 — 전환형 웹사이트

- 메뉴별 페이지, 채용 검색, 관심공고
- 의료인·병원 상담 신청
- 광고 상품·가격·주문 요청
- 모바일 반응형과 SEO

### Phase 2 — 운영 백오피스

- Supabase DB와 관리자 로그인
- 상담 리드, 병원, 의료인, 공고 관리
- 신청 이메일·알림
- 채용공고 등록/검수/게시

### Phase 3 — 회원 플랫폼

- 의사·병원 계정 및 인증
- 이력서/병원 서류 업로드
- 비공개 매칭과 지원 현황
- 찜·알림·마이페이지

### Phase 4 — 결제와 성장

- 토스페이먼츠 결제창·웹훅·환불
- 광고 자동 게시와 기간 관리
- 세금계산서/영수증 처리
- 광고 성과 리포트와 재구매

## Current Rules

Phase 2의 운영 도구 없이 공개 회원가입과 자동 결제를 먼저 열지 않습니다.
