# QUICK REF

## Purpose

제품의 핵심 결정을 빠르게 확인합니다.

## Current State

- 제품명: 메디헬퍼스
- 한 줄 정의: 의사의 비공개 이직과 병원의 의사 초빙을 직접 연결하는 의사 전문 헤드헌팅 플랫폼
- 핵심 고객: 이직·구직을 고민하는 의사와 의사를 채용하는 병원
- 핵심 전환: 의사 비공개 이직 상담 / 병원 의사 채용 의뢰 / 병원 초빙광고 상품 주문
- 수익모델: 채용광고, 프리미엄 노출, 핵심정보 열람권, 멤버십, 전담 헤드헌팅 계약
- 기본 채널: 모바일·PC 웹 링크
- 브랜드: 신뢰를 주는 딥 네이비, 연결을 뜻하는 블루, 케어를 뜻하는 민트

## Current Rules

첫 화면과 모든 페이지에서 전화·상담·광고 등록 중 하나의 다음 행동이 명확해야 합니다.

서비스 범위는 의사 채용과 이직에 집중합니다. 다른 의료직군 확장 문구나 오픈 알림은 전면 UI에 노출하지 않으며, 존재하지 않는 공고나 이용자 수를 만들어내지 않습니다.
